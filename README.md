# Hi Bean — 个人网站

> 线上地址：[www.chendouzi.com](https://www.chendouzi.com)

## 简介

Bean 的个人主页，包含粒子动画星空背景、实时数字时钟、个人形象展示、微信二维码弹窗以及自媒体/资料库/杂物间/创意工坊等导航链接。

## 技术栈

- **HTML5** + **CSS3**（响应式布局）
- **原生 JavaScript**（Canvas 粒子动画、数字时钟）
- **Google Fonts**（Noto Sans SC 思源黑体）
- 部署于 **Vercel**，绑定自定义域名 `www.chendouzi.com`

## 文件结构

```
个人网站/
├── index.html          # 主页面
├── style.css           # 样式表（含移动端响应式）
├── script.js           # Canvas 粒子动画 + 数字时钟逻辑
├── 人物形象.png         # 个人形象（透明背景）
├── 人物形象.jpg         # 个人形象原图
├── 二维码.jpg           # 微信二维码
├── wechat.png          # 微信按钮（人物形象 + 加个微信文字）
├── .gitignore
└── README.md           # 本文件
```

## 功能特性

- 🌌 **粒子动画星空背景**：Canvas 绘制动态粒子连线网络，随鼠标移动交互
- 🕐 **七段数码管数字时钟**：纯 CSS 实现的实时时钟
- 👧 **个人形象展示**：透明背景人物形象图，融入页面背景
- 💬 **微信二维码弹窗**：点击按钮显示 / 隐藏二维码
- 🔗 **导航链接**：自媒体（语雀）、资料库（腾讯文档）、杂物间（GitHub）、创意工坊（Bilibili）
